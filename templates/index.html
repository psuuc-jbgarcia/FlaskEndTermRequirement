Using loops (for, while) and conditionals (if, elif, else).

Creating reusable code blocks with parameters and return values.

Object-Oriented Programming (OOP)

Working with classes, objects, inheritance, and polymorphism.

# Example of OOP
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        return f"{self.name} makes a sound."

class Dog(Animal):
    def speak(self):
        return f"{self.name} barks."

dog = Dog("Rex")
print(dog.speak())  # Output: Rex barks.
Error Handling

Managing exceptions using try-except blocks.

try:
    # Code that might raise an exception
    result = 10 / 0  # ZeroDivisionError
except ZeroDivisionError:
    print("Cannot divide by zero!")
else:
    print("Division successful!")

Time Manipulation

Working with time using the datetime module and manipulating dates and times.

import datetime

# Define a time string in a specific format
time_string = "2025-01-02 14:30:00"
                        
# Parse the string into a datetime object using strptime
now = datetime.datetime.strptime(time_string, "%Y-%m-%d %H:%M:%S")
print("Parsed date and time:", now)
                        
# Add 5 days to the current date
new_date = now + datetime.timedelta(days=5)
print("New date after 5 days:", new_date)
Random Choice and randint

Using the random module to make random choices and generate random integers.

import random

# Randomly choose an item from a list
choices = ['apple', 'banana', 'cherry', 'date']
random_choice = random.choice(choices)
print("Randomly chosen fruit:", random_choice)

# Generate a random integer between 1 and 100
random_int = random.randint(1, 100)
print("Random integer between 1 and 100:", random_int)
